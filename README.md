# InfraAuto - 도면 기반 인프라 견적 자동산출

## v1 (현재 MVP)
- Paint 도면 PNG → 케이블/맨홀 검출 → 고정단가 견적 → Excel
- 직접 그리기 지원

---

## v2: 공종 분류 확장 + 단가 DB

### 공종 7종 (색상 코드 매핑)

| 공종 | 도면 색상 | 설명 |
|------|-----------|------|
| 관로 (conduit) | 검정 선 | 지하 관로 경로 |
| 케이블 (cable) | 파랑 선 | 케이블 포설 |
| 토공 (earthwork) | 갈색 선 | 굴착 경로 |
| 맨홀 (manhole) | 검정 큰 사각형 | 대형 맨홀 |
| 핸드홀 (handhole) | 검정 작은 사각형 | 소형 핸드홀 |
| 전주 (pole) | 빨강 원 | 전주 위치 |
| 접속함 (junction) | 녹색 사각형 | 접속함 |

### 단가 DB (SQLite)
- 공종/환경별 단가 관리
- 단가 편집 UI
